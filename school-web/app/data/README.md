# school-official-site

中学官网项目。

## 项目结构

- school-web：前台官网，使用 Nuxt
- school-admin：后台管理系统，后续开发
- school-api：后端接口服务，后续开发
- school-godot：Godot 互动模块，后续开发

## 当前阶段

已完成：

- 前台官网基础页面
- 首页
- 新闻列表页
- 新闻详情页
- 搜索页
- 学校信息配置
- 静态模拟新闻数据

## 本地运行

进入前台项目：

```powershell
cd school-web
npm install
npm run dev -- --host 127.0.0.1
```

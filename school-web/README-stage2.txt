第二阶段 Nuxt 学校官网组件化代码包

使用方法：
1. 解压本压缩包。
2. 将压缩包内的 app、public、package.json、nuxt.config.ts 复制到 D:\projects\school-official-site\school-web。
3. 如果提示是否覆盖，选择覆盖。
4. 在 VS Code 终端执行：
   cd D:\projects\school-official-site\school-web
   npm install
   npm run dev -- --host 127.0.0.1
5. 浏览器打开 http://127.0.0.1:3000/
